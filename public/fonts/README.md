# Self-hosted fonts

These WOFF2 files are downloaded from the official Google Fonts CSS API and
served locally to avoid the render-blocking Google Fonts stylesheet request.

- `Inter` is licensed under the SIL Open Font License 1.1.
- `Montserrat` is licensed under the SIL Open Font License 1.1.
- `Iceland` is licensed under the SIL Open Font License 1.1.

The matching license notices are stored in `public/fonts/licenses/` and are
copied into the production build with the font files.
